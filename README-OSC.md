# Unleash on Eyevinn Open Source Cloud

[Unleash](https://github.com/Unleash/unleash) is an open source feature management platform that gives you full control over your feature rollouts. Run it as a managed service on [Eyevinn Open Source Cloud](https://www.osaas.io) — no infrastructure to manage.

## About Eyevinn Open Source Cloud

[Eyevinn Open Source Cloud](https://www.osaas.io) (OSC) is a platform that makes it easy to run open source software as managed cloud services. Deploy and scale your favorite open source tools without the operational overhead.

## Getting Started

1. Sign in at [app.osaas.io](https://app.osaas.io)
2. Find **Unleash** in the service catalog
3. Provision a new instance — OSC handles the database, networking, and TLS
4. Access your Unleash dashboard at the provided hostname
